## Setup

1. Create a .env file with your `OPENAI_API_KEY` key in it.

```
OPENAI_API_KEY=youropenaikeyhere
```

2. run `pnpm i`

3. run `npm start`

4.  Open [http://localhost:3500/](http://localhost:3500/)
